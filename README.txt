Несколько программ и утилит на языках C (GTK), Javascript и bash:

c-gtk - две утилиты на C и библиотеке GTK:
  1. make_link - графическая утилита. Создает на рабочем столе Linux символьную ссылку на файл.
  2. smallpics - уменьшает выбранную папку с фотографиями (jpg). Опционально зипует уменьшенную папку в один архивный zip-файл (например, для того, чтобы удобно было передать по почте).

clearlogs - простая утилита по очистке логов на ОС Debian GNU/Linux

downloader - утилита для загрузки веб-сайтов на диск. Для создания веб-архива определённого сайта.

javascript_scriptmaker - утилита для преобразования html блоков кода в переменные javascript. Для последующей вставки в код веб-страницы.

keyboard_training - клавиатурный тренажёр. Имеет свою фичу. Помимо этого:
- тренирует моторику
- стрессоустойчивость
- восстанавливает самомнение о собственных навыках быстрой печати на клавиатуре до реального уровня.

learn_english - обучалка английскому. Помогает запоминать английские слова.

sokoban - моя реализация консольной игры-головоломки Sokoban
